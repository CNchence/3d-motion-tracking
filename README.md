# 3D Motion Track

Detect and track object via webcam in (semi) 3D fashion.

---

## Prerequisites

Ensure you have following installed:

  - [x] [gcc](https://gcc.gnu.org/) with C++11 support.
  - [x] [OpenCV 3.x](http://opencv.org/downloads.html)
  - [x] A webcamera !


## Compile and run

As simple as:

```
$ make
$ bin/track
```
